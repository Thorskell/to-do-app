# ✅ To‑Do List – Aplicação Web

Uma aplicação de lista de tarefas moderna, interativa e responsiva, desenvolvida com **HTML, CSS e JavaScript puro**, focada em **boa experiência do usuário (UX)**, **design limpo (UI)** e **código organizado**.

---

## 🖥️ Demonstração

📌 Aplicação roda diretamente no navegador  
📌 Não requer instalação ou bibliotecas externas

Basta abrir o arquivo `index.html`.

---

## 🚀 Funcionalidades

- ✅ Adicionar tarefas
- ✏️ Editar tarefas diretamente na lista
- ❌ Excluir tarefas com confirmação
- ✔️ Marcar tarefas como concluídas
- 🎯 Definição de prioridade:
  - Alta
  - Média
  - Baixa
- 🎨 Estilização condicional por prioridade
- 🌙 Modo claro / modo escuro
- 🔍 Filtros:
  - Todas
  - Pendentes
  - Concluídas
- 📊 Contador de tarefas:
  - Total
  - Concluídas
  - Pendentes
- 💾 Persistência de dados com `localStorage`
- ⌨️ Adicionar tarefa pressionando **Enter**
- ⚠️ Validação de entrada (evita tarefas vazias ou duplicadas)
- 🎞️ Animações suaves ao adicionar/remover tarefas

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
  - Flexbox
  - Variáveis CSS
  - Animações e transições
- **JavaScript (ES6+)**
  - Arrow functions
  - Manipulação do DOM
  - localStorage
- **Font Awesome**
  - Ícones da interface
